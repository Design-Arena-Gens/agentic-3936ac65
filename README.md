# AuraLedger Workspace

This repository contains the AuraLedger product prototype generated by the Codex agent. It includes a production-ready Next.js application and accompanying product documentation for an AI-driven personal expense management experience.

## Contents
- `ai-finance-manager/` — Next.js (App Router) implementation of the AuraLedger web experience
- `ai-finance-manager/docs/aura-ledger-brief.md` — product strategy pack with audience, UX flow, technical requirements, stack recommendations, and phased roadmap

## Quickstart
```bash
cd ai-finance-manager
npm install
npm run dev
# visit http://localhost:3000
```

Run checks before deploying:
```bash
npm run lint
npm run build
```

To generate exports directly from the UI use the Excel/PDF actions surfaced in the app header.

## Deployment
The app is optimised for Vercel (`vercel build` ready). Use the provided production command:
```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-3936ac65
```
